## REAR SHOCK PRESSURE DATA

### Fox FLOAT Shocks (Max 350 PSI, Sag 25-30%)
Starting point: PSI = rider weight in lbs. Adjust from there.
- Float DPS (smaller air can): runs ~5-10% lower PSI than rider weight
- Float X / DPX2: PSI roughly equals rider weight in lbs
- DHX Air: runs ~10% higher due to larger negative chamber

## MTB TYRE PRESSURE DATA

### Bike Faff Formula
Base: `pressure = (riderWeight / 5) + 15` (PSI)
Adjustments:
- 29" tyres: -4 to -8 PSI depending on width
- Non-tubeless: +4 PSI
- Rough terrain: +2 PSI
- XC: +1 PSI, DH: -1 PSI
- Front/rear split: rear = front + 2 PSI

### SILCA Approach
- Uses 15% tyre deflection model for road
- For MTB, uses impedance-optimised pressure (break point)
- Weight distribution: 45% front, 55% rear
- Tubeless: 5-10 PSI lower than clincher

### Recommended Calculator Approach
Use lookup tables per model for forks (Fox and RockShox have very different pressure curves).
For rear shocks, keep existing brand-specific lookup tables but verify against published data.
For tyres, use Bike Faff-style formula with terrain and casing adjustments.

## KEY FINDINGS

### Current Calculator Issues

1. **Fork pressure is brand-agnostic** — uses a single linear formula (`PSI = kg * multiplier + offset`) when in reality Fox 36 FLOAT at 175lbs = 86 PSI, while RockShox Zeb at 175lbs = 50-74 PSI depending on travel. This is a MASSIVE difference.

2. **No fork brand/model selection** — the current calculator only has shock brand/model selection, not fork brand/model.

3. **Shock data is reasonably close** but needs refinement:
   - Fox Float X/DPX2 existing: [100,140], [120,160], etc. vs Fox official: "PSI = body weight in lbs" starting point
   - RockShox Super Deluxe existing data matches the official chart well
   - DVO Topaz data matches well

4. **Tyre pressure calculator**: existing MTB tyre calc is reasonable but missing terrain type and casing inputs.

### What Needs to Change

1. Add fork brand and model selection (Fox 32/34/36/38/40, RockShox SID/Pike/Lyrik/Zeb/Boxxer, Marzocchi)
2. Use official manufacturer lookup tables for fork pressure with interpolation
3. Add volume spacer/token count as an input (affects air spring curve)
4. Add terrain type for tyre pressure (hardpack, loam, loose/rocky, mud)
5. Add tyre casing selector (lightweight, trail, enduro/DH)
6. Add rim internal width for MTB tyres
